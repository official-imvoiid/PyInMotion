How to Run This Code:
Clone the repo, navigate into the project folder, and run the script. For Windows, simply double-click UsernameGen.py—it should execute. For Linux (e.g., Kali, Arch), open a terminal in the project directory and run:

  python UsernameGen.py  
  or
  python3 UsernameGen.py

Make sure Python is installed on your system. No magic, just code.

What's Used in This Project:
random: To pick random words, numbers, and characters.
string: Provides ASCII letters and digits for random generation.
os: Checks if the configuration file exists and to interact with operating system.
json: Loads and saves custom word lists in JSON format.
datetime: Generates a timestamp for saved username files.
